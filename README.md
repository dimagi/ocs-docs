# Open Chat Studio Docs

## Quickstart

Assuming you've already cloned this repository:

1. Install UV

    ```shell
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```
    
    See https://docs.astral.sh/uv/getting-started/installation/

2. Set up the project
    
    ```shell
    uv venv
    uv sync
    ```

3. Start the project
    
    ```shell
    uv run mkdocs serve
    ```
